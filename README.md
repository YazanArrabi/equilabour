# EquiLabour

Monorepo for the EquiLabour recruitment and skill-matching system.

## Apps

- `apps/web` — React + Vite frontend
- `apps/api` — Express + TypeScript backend

## Packages

- `packages/shared` — shared code

## Stack

- React + Vite
- Node.js + Express
- PostgreSQL
- Prisma
- AWS S3
- Anthropic Claude API
